# Project Snapshot: V1.0
**Date:** 2026-01-26
**Description:** Stable single-ticker mechanical trading system with Google Finance / Stooq data integration and loading progress UI.

## Contents
- Real-time Price: Google Finance (via server.js)
- Historical Data: Stooq CSV (via server.js)
- UI: Premium Dark Dashboard with Loading Progress Bar
- Tickers Supported: TQQQ, SOXL, UPRO (Synced with advanced settings)
- Lock System: Prevents ticker change during active trades.

*This version is backed up in `/backups/v1.0/`*
